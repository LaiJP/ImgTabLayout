# ImgTabLayout
自定义可以设置图片的TabLayout

示例图片：https://github.com/zuoguiqing/ImgTabLayout/blob/master/155325956.jpg


添加依赖
一、
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
二、
  dependencies {
	        implementation 'com.github.zuoguiqing:ImgTabLayout:Tag'
	}
